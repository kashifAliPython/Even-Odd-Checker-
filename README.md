# Even or Odd Number checker 
## Description 
This is a simple python project that allows the user to check weather a given number is **even** or **odd**.
It is designed to practise basic **python concepts** such as input , conditionals , and modulo operations.

## Features 
- Take Input form the user
- Determine if the number is **even** or ***od**
- Display the result clearly to the user.

## How to use 
1. Run the python file 'Even-odd-checker.py'.
2. Enter any integer number when prompted.
3. The program will output weather the number is **even** or **odd**.


## Example Run
Enter a number : 8
          The Number you have Entered is Even: 8

Enter a number : 7
          The Number You have Entered is Odd: 7
